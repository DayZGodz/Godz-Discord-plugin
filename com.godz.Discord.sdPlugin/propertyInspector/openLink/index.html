Como criar um Bot?
  1. Acesse discord.com/developers/applications
  2. Selecione seu aplicativo (mesmo do Client ID)
  3. Va em Bot no menu lateral
  4. Clique em Reset Token e copie o token
  5. Ative Message Content Intent em Privileged Intents
  6. Va em OAuth2 > URL Generator, marque bot
  7. Em Bot Permissions marque Read Message History
  8. Copie a URL gerada e abra no navegador para adicionar o bot ao servidor
